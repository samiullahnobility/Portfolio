# Samiullah Portfolio

Modern personal portfolio website for Samiullah, a full-stack .NET, Angular, React, and Next.js developer focused on practical business software.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React components

## Features

- Premium dark portfolio UI
- Responsive navbar and mobile menu
- Hero section with profile photo and dashboard preview
- About, skills, projects, services, process, and contact sections
- Data-driven content in `src/data/portfolio.ts`
- Downloadable resume at `/resume.pdf`
- Demo contact form UI

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run lint
npm run build
```
